# Weather CLI

CLI прогноза погоды

## Использование

weather \<command>

|                       |                  |
|:----------------------|:-----------------|
| weather               | вывод погоды     |
| weather -h            | помощь           |
| weather -s \<city>    | установка города |
| weather -t \<api-key> | установка токена |
